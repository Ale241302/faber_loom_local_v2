### Caveats técnicos

- **Nexscope skills son texto plano (markdown instructions)**: No ejecutan código, no llaman APIs. Son frameworks de expertise que el LLM aplica. No reemplazan Helium 10, Jungle Scout, o SP-API. Para datos reales se necesita MCP server o skill con scripts Python.
- **Instalación nexscope-ai**: El comando `npx skills add nexscope-ai/Amazon-Skills -g` instala TODAS las 51 skills globalmente en `~/.claude/skills/`. Ocupan ~2MB de disco. Skills individuales con `--skill <name>`. El repo usa paths relativos desde Abril 2026.
- **Claude-ecom requiere Python 3.10+**: Instala venv privado en `~/.claude/skills/ecom/.venv`. Backend deterministico en Python, narrativa en Claude. Requiere CSV con columnas mínimas: order_id, order_date, customer_id, revenue. No conecta APIs.
- **MarceauSolutions/amazon-seller-mcp es MCP, no skill**: Funciona con Claude Desktop vía MCP server, no como skill de skills.sh. Requiere SP-API OAuth setup. Solo 1 star, 1 commit (Enero 2026) — proyecto muy temprano, riesgo de abandono. Pero es el ÚNICO con SP-API real encontrado.
- **MarketplaceAdPros/skill-amazon-ads requiere cuenta paga**: MCP server conectado a plataforma de pago. No es gratuito. Skill en sí es MIT.
- **Eronred/aso-skills requiere Appeeky API para datos reales**: Sin API key funciona con frameworks genéricos. Con Appeeky ($8/mes) obtiene datos App Store reales. Técnicas transferibles a Amazon pero dominio diferente.
- **affaan-m/everything-claude-code@inventory-demand-planning es genérico retail**: Diseñado para retailers 40-200 tiendas, Blue Yonder/SAP. No Amazon FBA-specific. Adaptable pero requiere context injection de FBA constraints.
- **Security audits nexscope**: `amazon-keyword-research` tiene Gen Agent Trust Hub Fail, Socket Warn, Snyk Warn. No son bloqueantes pero indican que el skill hace scraping web (Amazon autocomplete) lo que puede ser frágil.
- **No hay skills de Shopify-Mercado Libre bridge**: El ecosistema de skills.sh tiene 0 skills de Mercado Libre. Expansión a ML requiere skill custom.
- **Riesgo de skills.sh listings beta**: Muchas skills nexscope están marcadas Beta (🔶). Las Available (✅) son: amazon-keyword-research, amazon-listing-optimization, amazon-fba-calculator, amazon-ppc-campaign, amazon-sales-estimator, tariff-calculator-amazon.
