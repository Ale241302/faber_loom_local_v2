### Caveats técnicos

- **`timesfm-forecasting`**: Requiere ~1.5 GB RAM en CPU (~1 GB VRAM en GPU). El modelo de 500M params necesita ~32 GB RAM. Incluye preflight checker obligatorio. Snyk Warn en el audit de seguridad. No soporta multivariate forecasting directamente — es univariate. No tiene anomaly detection built-in, pero se puede derivar de quantiles.
- **`claude-ecom`**: Instalación manual (clonar repo, copiar a `.claude/skills/ecom/`). Requiere Python backend. No está empaquetado como skill de skills.sh (sin `npx skills add`). Última actualización marzo 2026.
- **`data-analysis-skill` (dongzhang84)**: Requiere Python 3.8+ con `pandas`, `openpyxl`, `tabulate`, `chardet` y Node.js 18+ con `puppeteer` y `pptxgenjs`. El export a PPTX usa Puppeteer para screenshots de HTML. No está en skills.sh — instalación manual.
- **`data-analytics-skills` (nimrodfisher)**: Colección de 31 skills portables. Instalación manual copiando los skills individuales a `.claude/skills/`. No hay un comando `npx skills add` para toda la colección. Algunos skills son prompts estructurados sin scripts.
- **`anomaly-detector` (majesticlabs)**: Incluye scripts Python con dependencias de `pandas`, `scikit-learn`. Es un skill ligero con 44 installs/semana y 37 stars — madurez limitada.
- **`amazon-seller` (claude-office-skills)**: Snyk Warn en audit. 89 stars. Es más un skill de "checklist/operaciones" que de análisis cuantitativo profundo. No tiene scripts de procesamiento de datos.
- **`charlie-cfo-skill`**: 222 installs totales. Es más un framework de consulta/guía que un skill con scripts de procesamiento. No conecta directamente a datos de ventas.
- **`csv-data-summarizer`**: Snyk Warn. Genera visualizaciones con matplotlib — calidad visual básica. No soporta Excel nativamente (solo CSV).
- **`scikit-survival` (davila7)**: 263 installs. Es un template/prompt para usar scikit-survival, no un skill con scripts pre-empaquetados. Requiere conocimiento de Python por parte del usuario.
- **`cohort-analysis` (phuryn)**: Skill de PM, no de e-commerce. Diseñado para product analytics genérico, no para cohorts de ventas Amazon.
- **Conflictos potenciales**: MWT ya tiene `anthropic-skills/xlsx` instalado. El skill `excel-automation` de `claude-office-skills` podría tener solapamiento. Recomendar solo si se necesita automatización avanzada (macros, VBA) que xlsx base no cubre.
- **Dependencias comunes**: Varios skills requieren `pandas`, `matplotlib`, `scikit-learn`. Instalar múltiples skills con dependencias Python puede causar conflictos de versión si no se usa un venv dedicado.
