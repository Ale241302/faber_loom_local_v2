FaberLoom Design System

Tokens, tipografía y widgets del control-plane. Paleta editorial-warm adyacente a Claude: coral sobre cream, con dark mode de graphite profundo. Tipografía trivial: Georgia serif itálica para display, Inter para UI, JetBrains Mono para IDs y trazas técnicas.
