The type system.

Three families, one job each. EB Garamond speaks — titles and editorial voice. Geist works — every interface surface. Geist Mono records — data, timestamps and the audit trail. The serif is the only voice; never set body copy or UI in it.

04

Rules of use

✓ Do
  • Garamond italic for the wordmark, titles and pull-quotes only.
  • Geist for all body, UI and controls — the workhorse.
  • Geist Mono for anything that must be exact: time, IDs, diffs, code.
  • Keep editorial measure at 60–75 characters.
  • Use tabular figures (Geist Mono) when numbers must align.
✕ Don’t
  • Never set body copy or buttons in Garamond — it breaks the editorial calm.
  • Never go below 12px in the interface, or 14px in print.
  • No upright Garamond for the wordmark — italic is the voice.
  • No third type family; three is the whole system.
  • Don’t letterspace lowercase Geist; reserve tracking for mono caps.
