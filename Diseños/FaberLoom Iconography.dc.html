FaberLoom · Iconography

The icon system.

Two families on one grid. A custom brand set carrying the trama / oficio DNA — woven threads, deliberate stitches, a single human anchor — and a standard UI set redrawn for cohesion. Every glyph: 24×24, stroke 1.75, round caps, currentColor + one optional coral accent.

04 · COLOR

Color in context

The glyph stays neutral; color carries the meaning. Default is ink (inactive). Coral is the one accent — action and AI. The functional states stay warm, never blue or purple. Each surface retunes the same roles for contrast.

05

Construction rules

✓Do
  • 24×24 viewBox, stroke 1.75, round caps + joins.
  • currentColor for strokes — the icon inherits text color.
  • One coral accent shape max, for an active / AI state.
  • A single filled node marks the human anchor or «en uso».
✕Don’t
  • No mixed stroke weights across the set (brand connectors aside).
  • No fills except the deliberate anchor node.
  • No second accent colour — coral is the only one.
  • Don’t shrink a brand glyph below 16px — simplify instead.
